**1. Install packages**<br />
```
    $ go get github.com/gorilla/websocket
    $ go get github.com/shirou/gopsutil/mem
```

**2. Run Go Websocket server**<br />
```
    $ cd go-ws-server
    $ go run 
```

**3. Start NodeJS Websocket client**<br />
```
    $ cd node-ws-client
    $ npm install
    $ node main.js        
```
